# Task List API

## Description

This is an API built on [Nest](https://github.com/nestjs/nest) framework using TypeScript. Here we offer you can add/edit/update any task per user, also we guarantee the list of the tasks with their status.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
